## Petition Flow An Ai Powered Path to Resolution

This project is a **Streamlit web application** for managing and classifying citizen petitions using **Machine Learning and Rule-Based NLP**.

---

## Features

- Generate **synthetic petition dataset** with department and urgency labels.
- Train ML models using **TF-IDF + Random Forest**.
- Intelligent fallback to rule-based classification when models are unavailable.
- Upload text, files, or audio petitions.
